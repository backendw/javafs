HTML DOM 문서

클릭해 보세요.

메서드

getElementById('아이디명') - ID로 HTML 요소 찾기

getElementsByClassName('클래스명') - 클래스 이름으로 HTML 요소 찾기

getElementsByTagName('태그명') - 태그 이름으로 HTML 요소 찾기

querySelector('선택자') - 선택자로 HTML 하나의 요소 찾기

querySelectorAll('선택자') - 선택자로 HTML 모든 요소 찾기

속성

element.innerHTML = HTML 내용

element.style.property = CSS 스타일 값

element.attribute = 'HTML 속성 값'

length - 요소의 수

HTML 이벤트 속성

onclick - 사용자가 마우스를 클릭할 때

onmouseover - 마우스가 요소 위로 이동할 때

onmouseout - 마우스가 요소 밖으로 벗어날 때

onmousedown - 마우스가 요소를 누르고 있을 때

onmouseup - 마우스가 요소를 눌렀다 떼었을 때
