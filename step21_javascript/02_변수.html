변수(Variables)

데이터를 저장하기 위한 컨테이너입니다(데이터 값 저장).

변수 선언 키워드는 var, let(값이 변경될 가능성 있다면), const가 있다.

변수는 고유한 이름으로 식별 되어야 합니다. 이러한 고유한 이름을 식별자라고 합니다.

변수(고유 식별자)의 이름을 구성하는 일반적인 규칙

  • 이름에는 문자, 숫자, 밑줄 및 달러 기호가 포함될 수 있습니다.
  • 이름은 문자로 시작해야 합니다.
  • 이름은 $ 및 _로 시작할 수도 있습니다.
  • 이름은 대소문자를 구분합니다(y와 Y는 다른 변수임).
  • 예약어(예: JavaScript 키워드)는 이름으로 사용할 수 없습니다.

명명 규칙

  • camelCase: JavaScript 자체, jQuery 및 기타 JavaScript 라이브러리에서 사용됩니다.
  • PascalCase: 종종 C 프로그래머가 선호합니다.
  • snake_case: 많은 프로그래머는 특히 SQL 데이터베이스에서 밑줄(under_scores)을 사용하는 것을 선호합니다. PHP 문서에서 자주 사용됩니다.
  • 하이픈은 빼기 시도로 오인될 수 있습니다. JavaScript 이름에는 하이픈을 사용할 수 없습니다.
  • $ 기호로 이름을 시작하지 마십시오. 많은 JavaScript 라이브러리 이름과 충돌하게 됩니다.

템플릿 리터럴(Template Literals, Back-Tics 구문)

따옴표("") 대신 역따옴표(``)를 사용하여 문자열을 정의합니다.

문자열 내에서 작은따옴표와 큰따옴표를 모두 사용할 수 있습니다.

여러 줄 문자열을 허용합니다.

문자열에 변수를 허용합니다. 변수 또는 표현식을 실제 값으로 자동 교체하는 것을 문자열 보간(Interpolation) 이라고 합니다 .
